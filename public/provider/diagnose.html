1

NVIDIAドライバ（CUDA対応版）をインストール

⚡ CUDAドライバとは？
NVIDIAのGPUでAI処理を高速化するために必要なソフトウェアです。
診断で「CUDAバージョン取得不可」が出た場合はここから解決できます。
1 NVIDIAドライバをダウンロードする
🟢 GeForce Game Ready Driver（推奨・CUDA対応）
最新のCUDA対応ドライバ · インストール後に再起動が必要
📥 ドライバをダウンロード
または nvidia.com/Download から型番を指定してダウンロードも可能
2 インストーラーを実行する
→ ダウンロードした setup.exe をダブルクリック
→ インストール方法は 「エクスプレスインストール」 を選択（推奨）
→ インストール完了後 PCを再起動 してください
3 インストールを確認する（再起動後）
コマンドプロンプトで以下を実行：
nvidia-smi
✅ 以下のような出力が表示されれば成功です：
Driver Version: 5xx.xx   CUDA Version: 12.x
その後、診断ページをリロードして「診断を開始する」を再度押してください。
❓ うまくインストールできない場合
古いドライバが残っている場合：
「コントロールパネル」→「プログラムのアンインストール」→「NVIDIA」関連を全て削除してから再インストール
GeForce Experienceを使う方法：
GeForce Experience をインストールすると「ドライバ」タブから自動更新できます
2

Node.js をインストール

💡 Node.jsとは？
Janctionサーバーを動かすために必要な実行環境です。一度インストールすれば次回以降は不要です。
1 Node.js をダウンロードする
🟢 Node.js 20 LTS（推奨版）
「LTS」と書いてある安定版を選んでください
📥 nodejs.org を開く
サイトを開いたら「LTS 推奨版」と書かれたボタンをクリックしてダウンロード
2 インストーラーを実行する
→ ダウンロードした node-v20.x.x-x64.msi をダブルクリック
→ 「Next」を繰り返し押して進む（設定は変えなくてOK）
→ 「Install」をクリック → インストール完了
→ インストール後、PCを再起動してください
💡「このアプリが変更を加えることを許可しますか？」→「はい」をクリック
3 インストールを確認する
PowerShellを開いて（管理者不要）、以下のコマンドを実行：
node --version
✅ v20.x.x のようにバージョンが表示されれば成功です！
「認識されていない」と出た場合は、PCを再起動してから再度お試しください。
3

Janctionエージェントをインストール・起動

💡 Janctionエージェントとは？
あなたのPCにインストールする常駐プログラムです。
GPU情報をリアルタイムでJanctionサーバーに送信し、ユーザーがあなたのGPUを利用できる状態に保ちます。
📌 ポートフォワーディング不要 · バックグラウンド動作 · 自動再接続
1 エージェントをダウンロードする
⚡ Janction Agent（Windows用）
v1.0.0 · 約10MB · インストール不要・ダブルクリックで起動
⬇️ ダウンロード
💡 ダウンロードしたファイルは「ダウンロード」フォルダに保存されます
2 ダブルクリックで起動する
→ ダウンロードした janction-agent.exe をダブルクリック
→ 「WindowsによってPCが保護されました」が出た場合：
「詳細情報」→「実行」をクリックしてください
✅ 黒いウィンドウ（⚡ Janction Agent）が起動すれば成功です
3 メールアドレスを入力して接続
→ Janctionに登録したメールアドレスを入力
→ 「🚀 接続して収益化を開始」ボタンをクリック
✅ 「接続完了！」のメッセージが出れば、あなたのGPUが公開されます！
エージェントはバックグラウンドで動き続けます。
① リポジトリをクローン
git clone https://github.com/taichiyao333/janction.git
cd janction && npm install
npm start
🔌 エージェント起動状態
「確認する」を押すとエージェントの起動状態をチェックします
4

GPUを登録する

エージェント起動後、プロバイダーポータルからGPUを登録すれば完了です。
