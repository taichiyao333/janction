Requirements

ホストになるための
動作環境

特別な機材は不要。現在お使いのゲーミングPCやワークステーションがそのまま使えます。

✅ 必要な環境

  • NVIDIA GPU（GTX1080以降またはRTX系）搭載PC
  • Windows 10/11 または Ubuntu 20.04+
  • RAM 8GB以上（推奨16GB）
  • SSD/HDD 空き容量 50GB以上
  • 上り100Mbps以上の固定回線（推奨500Mbps）
  • 常時接続できる電源環境（UPS推奨）

⚠️ 注意事項

  • 貸し出し中は電力消費が増加します（目安：RTX4090で450W追加）
  • GPUの熱管理（ケースエアフロー）が重要です
  • ノートPCはバッテリー・熱設計の関係で非推奨
  • AMD GPU（Radeon）は現在非対応
  • マイニング目的の利用は禁止です
  • 18歳以上または保護者の同意が必要
