⚙️ 必要な環境・条件

以下の条件を満たしているか確認してください。

⬜
NVIDIA GPU（GTX 1060以上推奨）
RTX 3090・4090・A4500・A100など。AMD GPUは非対応。
✅
Windows 10/11 または Ubuntu 20.04+
64bit OS。macOSは非対応。
⬜
NVIDIA Driver 520以上
nvidia-smi で確認できる状態。
⬜
Node.js 18以上インストール済み
nodejs.org からダウンロード。
⬜
常時接続インターネット環境
上り帯域10Mbps以上推奨。光回線が理想的。

🌐 インターネットに公開する

Cloudflare Tunnelを使って自宅PCを外部から安全にアクセス可能にします。

A
cloudflared をインストール
PowerShell (管理者)
winget install Cloudflare.cloudflared
B
サーバーをトンネルで公開
PowerShell
# GPUサーバーを起動
cd F:\antigravity\gpu-platform
node server/index.js
# 別ウィンドウでトンネル起動
cloudflared tunnel --url http://localhost:3000
C
公開URLをプラットフォームに登録
自動発行されるURLを管理者に連絡するか、固定ドメインを設定してください。
⚠️ セキュリティ注意: Cloudflare Tunnelは暗号化されますが、サーバーのJWT_SECRETを必ず変更してください。
